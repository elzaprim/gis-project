# Palm Oil Deforestation Risk Analysis

This project analyzes deforestation risk in the palm oil supply chain using
forest cover and forest loss data.

The analysis uses the Hansen Global Forest Change dataset to estimate
environmental risk levels for:

- Estates
- Mills
- Vendors

## Data Source

Hansen Global Forest Change Dataset  
https://storage.googleapis.com/earthenginepartners-hansen/GFC-2024-v1.12/download.html

Only tiles covering Indonesia were downloaded and processed.
